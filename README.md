# XiaMiMusic
基于nodejs+electron的音乐播放器（仿虾米音乐）

打开方式：

1.全局安装electron
    ```
    npm install electron -g
    ```
    
2.```git clone https://github.com/CleverFan/XiaMiMusic```

3.```cd XiaMiMusic```

4.```npm install & npm start```
